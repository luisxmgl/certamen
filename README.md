# Portafolio Luis I.

#Este repositorio contiene mi portafolio académico y profesional, desarrollado como parte de la asignatura de Ingeniería en Informática. El objetivo es mostrar mis habilidades técnicas, proyectos realizados y un formulario de contacto funcional, aplicando buenas prácticas de HTML5 y CSS3.

#-Objetivos del proyecto:

- Presentar información personal y académica de manera clara y estructurada.
- Demostrar el uso de etiquetas semánticas de HTML5.
- Aplicar estilos modernos con CSS3, incluyendo Flexbox y Grid.
- Cumplir con los requisitos de commits solicitados por la profesora.

#-Estructura del sitio:
-Encabezado o Header
- Logo personal y nombre.  
- Barra de navegación con enlaces internos a cada sección.  

-Sobre mí:
   -Presentación breve con foto.  
   -Artículo con descripción personal y cita inspiradora.  
   -Aside con datos personales en lista desordenada.

-Habilidades:
   -Tabla con tecnologías, nivel y barras de progreso.

-Proyectos.
  
-Contacto:
   -Formulario con etiquetas requeridas.  
   -Aside con información de contacto (email, ciudad, teléfono, GitHub).  

-Footer:
   - Logo circular con iniciales.  
   - Nombre completo.  
   - Ícono de GitHub enlazado a mi perfil.  
   - Derechos reservados.  


#-Tecnologías utilizadas:

- HTML5: estructura semántica y etiquetas modernas.  
- CSS3: estilos responsivos, Flexbox y Grid.  
- Git / GitHub: control de versiones y commits documentados.
